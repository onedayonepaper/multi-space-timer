# 멀티 공간 타이머 (Multi Space Timer)

**어떤 공간이든 시간 관리 한 번에**

스터디카페, 코인세탁방, 공유오피스 등 시간 단위로 운영되는 공간을 위한 간편한 타이머 관리 솔루션입니다.

---

## 주요 특징

### v1.0 (최신) 🎉
- ⚙️ **관리자 모드** - PIN 인증, 설정 변경, 데이터 관리
- 📋 **업종 템플릿** - 스터디카페/코인세탁/공유오피스 등 7가지 템플릿
- 🔔 **사운드 알림** - 만료/경고 알림음 (ON/OFF 가능)
- 📊 **사용 통계** - 공간별 사용 기록 및 인기 공간 분석
- 💾 **백업/복원** - 설정 및 데이터 내보내기/불러오기

### 기본 기능
- ✅ **최대 24개 공간 관리** - 8/12/16/20/24개 선택 가능
- ⏱️ **직관적인 타이머** - 업종별 프리셋 시간 커스터마이징
- 💾 **오프라인 우선** - 인터넷 없이도 작동, localStorage 자동 저장
- 📱 **태블릿 최적화** - 안드로이드 태블릿, PC 모두 지원
- 🔄 **자동 상태 업데이트** - 실시간 남은 시간 표시, 만료/경고 알림
- 🎨 **키오스크 스타일** - 큰 버튼, 명확한 시각적 구분

프로젝트 파일을 다운로드하거나 클론합니다:

`index.html` 파일을 더블클릭하거나 브라우저로 드래그하여 실행합니다.

**추천 브라우저**: Chrome, Edge, Safari

## 사용 방법

### 기본 사용법

1. **공간 선택**
   - 화면에 표시된 16개 공간 중 하나를 클릭합니다

2. **시간 설정**
   - 팝업 모달에서 1/2/3시간 중 하나를 선택합니다
   - 타이머가 자동으로 시작됩니다

3. **상태 확인**
   - **미사용** (회색): 사용 중이지 않은 공간
   - **사용 중** (파란색): 타이머 동작 중, 남은 시간 표시
   - **만료됨** (빨간색): 시간이 종료된 공간

4. **공간 초기화**
   - 공간을 다시 클릭하여 "공간 초기화" 버튼 클릭
   - 미사용 상태로 돌아갑니다

### 남은 시간 표시

- **1시간 미만**: `MM:SS` 형식 (예: `45:30`)
- **1시간 이상**: `H:MM 남음` 형식 (예: `2:15 남음`)
- **경고 상태**: 5분 전부터 노란색 경고 표시
- **만료 상태**: 빨간색 + "만료" 배지

### 관리자 모드 (v1.0 신규)

1. **관리자 모드 접근**
   - 우측 상단 톱니바퀴 아이콘 클릭
   - 기본 PIN: `0000` 입력

2. **업종 템플릿 선택**
   - 7가지 템플릿: 커스텀, 스터디카페, 코인세탁, 공유오피스, 수면카페, PC방, 노래방
   - 클릭 한 번으로 업종별 최적화된 설정 적용

3. **설정 변경**
   - 공간 개수: 8/12/16/20/24개 선택
   - 공간 이름: "좌석", "세탁기", "회의실" 등 커스텀
   - 사운드 알림: ON/OFF

4. **데이터 관리**
   - **설정 내보내기**: JSON 파일로 백업
   - **설정 불러오기**: 백업 파일에서 복원
   - **전체 초기화**: 모든 설정 및 데이터 삭제

## 업종별 활용 예시

### 스터디카페

- **열람석 시간제 관리**: 입장 시 자리 배정 + 타이머 시작
- **스터디룸 관리**: 룸별 예약 시간 카운트다운
- **프리셋**: 1/3/5시간 (기본 1/2/3시간 사용 또는 커스터마이징)

### 코인세탁방

- **세탁기/건조기 관리**: 각 기계별 남은 시간 표시
- **프리셋**: 30/40/60분 (main.js에서 수정 가능)
- **손님 알림**: 대기 공간 태블릿으로 진행 상황 확인

### 공유오피스

- **회의실 현황판**: 로비 모니터에 회의실 사용 현황 표시
- **폰부스/집중석**: 짧은 시간 이용 공간 관리
- **프리셋**: 30/60/120분

## 설치 가이드 (업장 배포용)

### 안드로이드 태블릿 설치

1. **Chrome 브라우저 설치**
   - Google Play 스토어에서 Chrome 다운로드

2. **파일 업로드**
   - 태블릿에 프로젝트 파일 업로드 (USB, Google Drive 등)
   - 또는 웹 서버에 업로드 후 URL 접속

3. **키오스크 모드 설정**
   - Chrome 주소창에 `index.html` 경로 입력
   - 설정 > 사이트 설정 > 전체화면 허용
   - 홈 화면에 바로가기 추가

4. **자동 실행 설정 (선택사항)**
   - 'Fully Kiosk Browser' 등 키오스크 앱 사용
   - 부팅 시 자동 실행 + 터치 잠금 설정

### PC (Windows/Mac) 설치

1. **Chrome 브라우저 실행**

2. **파일 열기**
   - 프로젝트 폴더의 `index.html` 파일을 브라우저로 드래그

3. **전체화면 모드** (F11)
   - 키오스크 스타일로 사용하려면 전체화면 권장

4. **자동 실행 설정 (Windows)**
   - Chrome 단축키 만들기: `chrome.exe --kiosk "file:///C:/경로/index.html"`
   - 시작 프로그램에 등록

## 커스터마이징

### 관리자 모드를 통한 설정 (권장)

v1.0부터는 **관리자 모드**에서 모든 설정을 쉽게 변경할 수 있습니다:

1. 톱니바퀴 아이콘 → PIN 입력 (`0000`)
2. 업종 템플릿 선택 또는 직접 설정
3. 저장 버튼 클릭

### 고급 설정 (개발자용)

코드 레벨에서 커스터마이징하려면:

**설정 파일**: `src/core/config.js`
```javascript
export const DEFAULT_CONFIG = {
  spaceCount: 16,
  presets: [1, 2, 3],
  labels: { spaceName: '공간', unit: '번' },
  // ...
};
```

**템플릿 추가**: `src/templates/templates.js`
```javascript
export const TEMPLATES = {
  myTemplate: {
    name: '내 템플릿',
    config: { /* 설정 */ }
  }
};
```

## FAQ

### Q. 인터넷이 없어도 작동하나요?
A. 네, localStorage 기반이라 오프라인에서도 완벽히 작동합니다.

### Q. 데이터가 날아가지 않나요?
A. 브라우저 캐시를 지우지 않는 한 데이터가 유지됩니다. 중요 데이터는 v2.0 클라우드 버전 사용을 권장합니다.

### Q. 여러 태블릿에서 동기화되나요?
A. 현재 버전은 단일 디바이스 전용입니다. 다지점 동기화는 v2.0에서 지원 예정입니다.

### Q. 가격은 어떻게 되나요?
A. 현재 버전은 무료 오픈소스입니다. Pro 버전은 추후 공지 예정입니다.
